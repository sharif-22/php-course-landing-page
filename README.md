## Installation

<h3>Clone the Repository:</h3>

```
git clone https://github.com/sharif-22/php-course-landing-page/tree/main
```

Navigate to the project

```
cd php-course-landing-page
```

Install the Dependencies

```
npm install
```

Run the Project

```
npm run dev
```
